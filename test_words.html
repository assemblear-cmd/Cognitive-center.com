Субтест 5: Словарный

Этот субтест оценивает общий запас слов, уровень развития речи и способность к абстрактному мышлению.

Как проводить тест?
  • Начало теста:
    • Для детей 5-8 лет (или при подозрении на умственное снижение) начинайте с задания №5.
    • Для детей старше 8 лет начинайте с задания №10.
  • Правило возврата:
    • Для детей 5-8 лет: Если испытуемый не справился с заданием №5 (0 баллов), вернитесь к заданиям №1, 2, 3, 4. Если задание №5 выполнено, засчитывается "аванс" (4 балла) за задания №1-4.
    • Для детей 8+ лет: Если испытуемый не справился с заданиями №10, №11 и №12 (все по 0 баллов), вернитесь к заданиям с №9 в обратном порядке. Если выполнено хотя бы одно из заданий №10-12, засчитывается "аванс" (14 баллов) за задания №1-9.
  • Прекращение теста: Тестирование прекращается после 5 неудач (ответов на 0 баллов) подряд.
  • Оценка:
    • 2 балла: Полное, точное определение, отражающее основной смысл слова.
    • 1 балл: Частично верное определение, указание на конкретное использование или менее существенный признак.
    • 0 баллов: Неверное определение.
    • Задания 1-4: Оцениваются только в 0 или 1 балл.

Заметки для интерпретации:

Выполнение этого субтеста зависит от широты кругозора, начитанности, любознательности и способности к обучению. Высокие оценки свидетельствуют о богатом словарном запасе и хорошо развитом вербальном интеллекте. Низкие оценки могут быть связаны с ограниченным кругозором, педагогической запущенностью или специфическими речевыми трудностями.
